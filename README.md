# Walmart Sales Forecasting & Business Analytics Dashboard

## Project Overview

This project analyzes Walmart sales data to identify business trends, customer behavior, product performance, and future sales opportunities. The project combines Python-based data analysis with an interactive Power BI dashboard for business decision-making.

## Objectives

* Analyze historical sales performance
* Identify sales trends and patterns
* Understand customer and product behavior
* Forecast future sales using predictive models
* Build an interactive business dashboard

## Tech Stack

* Python
* Pandas
* NumPy
* Matplotlib
* Power BI
* Excel

## Dataset

Walmart Sales Dataset containing:

* Order Date
* Product Category
* Product Name
* Quantity
* Unit Price
* Total Sales
* Customer Type
* Store Location

## Exploratory Data Analysis (EDA)

Performed:

* Daily Sales Trend Analysis
* Weekly Sales Trend Analysis
* Monthly Sales Trend Analysis
* Sales Distribution Analysis
* Correlation Analysis
* Outlier Detection using Boxplots

## Forecasting Models

* Moving Average Forecasting
* Linear Regression Forecasting
* ARIMA Forecasting
